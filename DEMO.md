## 🎨 UI 特效

### 动画效果
- **浮动动画**: Logo和图标轻柔上下浮动
- **脉冲效果**: 按钮和重要元素发光脉冲
- **弹跳进入**: 页面元素弹跳式出现
- **渐变移动**: 背景渐变色彩流动
- **星星旋转**: 装饰性星星旋转动画

### 视觉设计
- **渐变背景**: 红色主题渐变，动态效果
- **玻璃态**: 半透明毛玻璃效果卡片
- **阴影系统**: 多层次阴影，增强立体感
- **稀有度配色**: 
  - 普通: 灰色
  - 稀有: 蓝色
  - 史诗: 紫色
  - 传说: 金色（脉冲发光）

## 🛠️ 技术栈

- **前端**: React 18 + TypeScript + Vite + Tailwind CSS
- **后端**: Node.js + Express + TypeScript
- **数据库**: SQLite (better-sqlite3)
- **认证**: JWT
- **部署**: Zeabur

## 📝 使用说明

### 用户流程
1. **注册/登录**: 创建账号或登录
2. **答题**: 回答红色精神问题获得积分（每题10-15分）
3. **集卡**: 花费50积分抽卡，收集8张卡片
4. **游戏**: 玩小游戏轻松赚积分
5. **兑换**: 积分兑换奖品，获取兑换码

### 管理员
- 访问 `/admin` 路径
- 输入 `ADMIN_KEY` 环境变量值
- 可以导入兑换码、查看统计数据
